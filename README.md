# MyFlixAngularClient

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.2.19.

## Objective
Build the client-side for the myFlix application using Angular, providing a single-page, responsive interface that interacts with an existing REST API and database.

## Project Overview
myFlix is a movie app for enthusiasts to explore information about movies, directors, and genres, and manage their favorite movies. This Angular client communicates with the myFlix server-side API, enabling users to register, log in, browse movies, and update their profiles.

## Features
- User registration and login forms
- Welcome view for authentication
- View all movies after login
- Single movie view with detailed info
- Director and genre views accessible from each movie
- Add/remove movies to/from favorites
- Edit or delete user profile
- Responsive design for all devices
- Built with Angular Material

## User Stories
- As a user, I want to receive information on movies, directors, and genres.
- As a user, I want to create a profile and save favorite movies.

## Technical Requirements
- Angular 9+
- Node.js & npm (latest)
- Angular Material for UI
- Typedoc comments in codebase
- JSDoc technical documentation
- Hosted on GitHub Pages

## Getting Started
1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/myFlix-Angular-client.git
    cd myFlix-Angular-client
    ```
2. **Install dependencies:**
    ```bash
    npm install
    ```
3. **Run the app locally:**
    ```bash
    npm start
    ```
4. **Build for production:**
    ```bash
    ng build --prod
    ```
